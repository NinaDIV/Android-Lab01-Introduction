# Android Lab 01 - Introduccion a Android

Primer laboratorio del curso de desarrollo Android. Cubre los conceptos fundamentales del ecosistema Android: configuracion del entorno, primera Activity y ciclo de vida basico.

## Temas cubiertos
- Configuracion de Android Studio
- Creacion de la primera Activity
- Layouts con XML: ConstraintLayout y LinearLayout
- Manejo de eventos basicos (onClick, onTextChanged)
- Ciclo de vida de Activity

- **Lenguaje:** Kotlin
- **SDK minimo:** API 24 (Android 7.0)

## Ejecutar el proyecto
1. Clona el repositorio
2. Abre en Android Studio
3. Conecta un dispositivo o inicia un emulador
4. Ejecuta con `Run > Run app`
