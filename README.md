# Laboratorio Android 01 — Introducción al Desarrollo Android con Kotlin

![Kotlin](https://img.shields.io/badge/Kotlin-7F52FF?style=flat&logo=kotlin&logoColor=white)
![Android](https://img.shields.io/badge/Android-3DDC84?style=flat&logo=android&logoColor=white)
![Android Studio](https://img.shields.io/badge/Android%20Studio-346AC1?style=flat&logo=androidstudio&logoColor=white)

Primer laboratorio del curso de desarrollo Android. Cubre los conceptos fundamentales del ecosistema Android: configuración del entorno de trabajo, creación de la primera Activity y el ciclo de vida básico de una aplicación. El repositorio incluye la guía de laboratorio en formato Word y PDF.

## 📚 Temas cubiertos

- Configuración de Android Studio
- Creación de la primera Activity
- Layouts con XML: `ConstraintLayout` y `LinearLayout`
- Manejo de eventos básicos (`onClick`, `onTextChanged`)
- Ciclo de vida de una Activity

## 🛠️ Tecnologías

| Tecnología | Detalle |
|---|---|
| Lenguaje | Kotlin |
| SDK mínimo | API 24 (Android 7.0 Nougat) |
| IDE | Android Studio |
| Layouts | XML (`ConstraintLayout`, `LinearLayout`) |

## ✅ Requisitos previos

Antes de empezar, asegúrate de tener instalado:

- **Android Studio** (versión reciente; incluye el JDK embebido necesario para compilar)
- **Android SDK** con la plataforma **API 24 o superior** (se instala desde el SDK Manager de Android Studio)
- Un **emulador Android** configurado en el AVD Manager **o** un **dispositivo físico** con la depuración USB activada
- **Git** para clonar el repositorio

## 🚀 Instalación y ejecución

1. Clona el repositorio:

   ```bash
   git clone https://github.com/NinaDIV/Android-Lab01-Introduction.git
   cd Android-Lab01-Introduction
   ```

2. Abre el proyecto en **Android Studio** (`File > Open`).
3. Conecta un dispositivo físico o inicia un emulador desde el **AVD Manager**.
4. Ejecuta la aplicación con `Run > Run app` (o el atajo `Shift + F10`).
5. Deberías ver la primera Activity del laboratorio desplegada en el dispositivo o emulador.

> 💡 La guía paso a paso del laboratorio está en los archivos `GLAB - S1 - BPAREJA - 2024-02` (disponible en `.docx` y `.pdf`) en la raíz del repositorio.
